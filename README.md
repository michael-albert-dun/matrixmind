# Matrixmind

A 2d-version of the classic Mastermind game.

Intended for desktop only. The player is presented with a grid which they fill with colours.

The computer has a secret filled version of the grid.

Each row and each column are scored as in Mastermind showing the number of pegs of the correct colour and position, and among the remainder, the number of the correct colour (but incorrect position). Previous guesses are shown as a sequence of somewhat smaller grids.
