# Two-dimensional mastermind

Two-dimensional mastermind. Fill the board with colours and then hit return to make that guess. Rinse and repeat.

## Controls

- Return to  make a guess (currently not checking that you're doing so from a filled board)
- Select a palette colour and then click on the board to make a square of that colour
- Or enter the colour number and the selected tile on the board will become that colour

## Results display

- Left and bottom display number of exact matches in the corresponding row and column
- Right and top display number of near-matches in the corresponding row and column
